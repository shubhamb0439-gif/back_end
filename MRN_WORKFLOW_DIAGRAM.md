## Continuous Detection Flow

```
┌─────────────────────────────────────────────────────────────────┐
│              New Transcript Arrives (final=true)                │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                Clear Existing Timer                             │
│          clearTimeout(state.mrnAutomationTimer)                 │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│           Set New Timer (500ms debounce)                        │
│    state.mrnAutomationTimer = setTimeout(...)                   │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼ (after 500ms)
┌─────────────────────────────────────────────────────────────────┐
│              continuousMRNDetection()                           │
│         - Load last 5 transcript items                          │
│         - Scan each for MRN (newest first)                      │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
                  ┌──────────────┐
                  │  MRN Found?  │
                  └──────┬───────┘
                         │
            ┌────────────┼────────────┐
            │ No                      │ Yes
            ▼                         ▼
    ┌───────────────┐      ┌──────────────────────┐
    │  End Scan     │      │  Check if Different  │
    └───────────────┘      │  from lastProcessed  │
                           └──────────┬───────────┘
                                      │
                         ┌────────────┼────────────┐
                         │ Same                    │ Different
                         ▼                         ▼
                 ┌───────────────┐      ┌──────────────────────┐
                 │  Skip         │      │ automateEHRWorkflow()│
                 └───────────────┘      │ (See main flow above)│
                                        └──────────────────────┘
```

## MRN Normalization Pipeline

```
┌─────────────────────────────────────────────────────────────────┐
│                   Raw Input String                              │
│            "M R N dash AB123" (spoken text)                     │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                   Step 1: Basic Cleanup                         │
│         - Trim whitespace                                       │
│         - Convert to UPPERCASE                                  │
│         - Normalize spaces (multiple → single)                  │
│         Result: "M R N DASH AB123"                              │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                Step 2: Spoken Word Replacement                  │
│         - Replace "DASH" → "-"                                  │
│         - Replace "HYPHEN" → "-"                                │
│         - Normalize hyphens (spaces around → single)            │
│         Result: "M R N - AB123"                                 │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                  Step 3: Pattern Matching                       │
│         Pattern: /\bM\s*R\s*N\s*-?\s*([A-Z0-9]{3,12})\b/       │
│         Captures: "AB123"                                       │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                 Step 4: Code Validation                         │
│         - Remove all spaces from code                           │
│         - Check length: 3-12 characters                         │
│         - Check format: alphanumeric only                       │
│         Code: "AB123" ✓                                         │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                Step 5: Format Assembly                          │
│         Template: "MRN-{CODE}"                                  │
│         Final Result: "MRN-AB123"                               │
└─────────────────────────────────────────────────────────────────┘
```

## Duplicate Prevention Logic

```
┌─────────────────────────────────────────────────────────────────┐
│               MRN Detected: "MRN-AB123"                         │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│          Check: state.lastProcessedMrn                          │
└────────────────────────┬────────────────────────────────────────┘
                         │
            ┌────────────┼────────────┐
            │                         │
            ▼                         ▼
   ┌────────────────┐         ┌──────────────────┐
   │  === null      │         │  === "MRN-AB123" │
   │  (first time)  │         │  (duplicate)     │
   └────────┬───────┘         └──────────┬───────┘
            │                            │
            ▼                            ▼
   ┌────────────────┐         ┌──────────────────┐
   │ TRIGGER        │         │ SKIP             │
   │ Automation     │         │ Already processed│
   └────────┬───────┘         └──────────────────┘
            │
            ▼
   ┌────────────────────┐
   │ Set lastProcessed  │
   │ = "MRN-AB123"      │
   └────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────────┐
│        Later: Different MRN Detected: "MRN-XYZ789"              │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│     Check: state.lastProcessedMrn === "MRN-AB123"               │
│            New MRN: "MRN-XYZ789"                                │
│            Result: Different! ✓                                 │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                TRIGGER New Automation                           │
│         Update lastProcessed = "MRN-XYZ789"                     │
└─────────────────────────────────────────────────────────────────┘
```

## Error Handling Flow

```
┌─────────────────────────────────────────────────────────────────┐
│              automateEHRWorkflow("MRN-AB123")                   │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                 try {                                           │
│         - Validate elements exist                               │
│         - Open sidebar                                          │
│         - Fill input                                            │
│         - Trigger search                                        │
│         - Wait for results                                      │
│         - Click Summary tab                                     │
│                 }                                               │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
                  ┌──────────────┐
                  │  Success?    │
                  └──────┬───────┘
                         │
            ┌────────────┼────────────┐
            │ Yes                     │ No (Error)
            ▼                         ▼
┌──────────────────────┐    ┌──────────────────────┐
│ Clear inProgress     │    │ catch (err) {        │
│ Automation Complete  │    │   console.warn()     │
└──────────────────────┘    │   Clear inProgress   │
                            │ }                    │
                            └──────────┬───────────┘
                                       │
                                       ▼
                            ┌──────────────────────┐
                            │ Continue Normal Flow │
                            │ (No User Error)      │
                            └──────────────────────┘
```
