🎯 Key Features

  • Single Button Access: PTT button controls all voice interactions
  • Press & Hold: Start recording on mousedown/touchstart
  • Voice Reactive Orb: Waveform animation responds to voice input
  • Auto Transcription: Real-time speech-to-text while holding
  • Voice Commands: Detect commands like "start stream", "stop stream"
  • Auto Connect: Connects to signaling server on page load
  • Response Playback: Shows audio responses from assistant
  • Keyboard Support: Spacebar also works as PTT

🔄 Workflow

  1. Page Load: Auto-fetches XR ID and connects to server
  2. Auto-Stream: When desktop connects, video stream starts automatically
  3. Press PTT: User presses and holds the mic button
  4. Recording Starts: Speech recognition activates, mic mutes (frees hardware)
  5. Live Waveform: Orb shows real-time audio visualization
  6. Real-time Transcript: Interim results display in orb
  7. Release PTT: Speech recognition stops, final transcript processed
  8. Command Detection: If voice command detected, execute it
  9. Transcript Routing: If not a command, send to scribe cockpit
  10. Response: Audio response shows in card with play button

🎤 Voice Commands

  • "start stream" - Start video streaming
  • "stop stream" - Stop video streaming
  • "mute" - Mute microphone
  • "unmute" - Unmute microphone
  • "connect" - Connect to server
  • "disconnect" - Disconnect from server
