# Orb UI Image Assets

The orb-based UI requires two image files that should be placed in this directory:

## Required Images

1. **orb-visual.png**
   - Large circular orb with animated waveform
   - Dimensions: ~600x600px recommended
   - Shows voice activity visualization
   - Source: User provided image (large orb with blue/purple waveform)

2. **mic-button.png**
   - Microphone button icon
   - Dimensions: ~200x200px recommended
   - Circular button with microphone icon
   - Source: User provided image (small circular mic button)

## Image Placement

Place the images with these exact names:
- `/frontend/public/images/orb-visual.png`
- `/frontend/public/images/mic-button.png`

## Note

The user has provided these images in the chat. They need to be saved manually to the correct locations above.
